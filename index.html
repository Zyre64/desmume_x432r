DeSmuME X432R

This is a customized version of DeSmuME.
Added experimental high-resolution rendering features.
It's poorly debugged, so use it at the risk of crashing.
Only built for Windows.
-> Origional History File <-

The mysterious old Japanese X432R fork has some frontend features that may be useful for some users. It is very outdated now. It is available only for Windows 32-bit x86 and Windows 64-bit x86-x64.
